Contributing to Life Calendar

Building a website like this is a lot of work, but we believe that the end result is worth it. However, this site is currently being primarily maintained and built by one person - Simon Ilincev - who'd appreciate co-contributors. It's easy to help with the publicy available todo list.

This site is built with Python and the Flask library and vanilla JavaScript with ES6 standards. Both the backend and the frontend are open to contributions, and you can easily open a pull request or fork the project on GitHub here.

If you aren't well-versed with programming, but would still like to help keep the site running, you can support it financially. This site has no advertisements, premium subscription, and doesn't sell data. While that makes it the best it can be for its users - which is and always has been the goal - it also means that there's no generated source of money for site + database hosting and the domain name.
